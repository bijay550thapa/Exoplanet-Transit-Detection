# Exoplanet Transit Detection using Kepler Data

This project demonstrates the detection of an exoplanet using archival data from the Kepler Space Telescope.

## Method
- Light curve analysis using Python
- Transit detection using Box Least Squares (BLS)
- Phase folding to confirm transit signal

## Results
- Orbital period: ~2.2047 days
- Transit depth: ~0.0066
- Clear transit signal observed

## Conclusion
This project successfully identifies an exoplanet transit signal consistent with a hot Jupiter.
